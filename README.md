Diabetes Prevalence Chatbot (AI + Public Health Project)

This project analyzes U.S. diabetes prevalence data using the CDC BRFSS dataset and builds an AI chatbot that allows users to request diabetes statistics based on their geographical location state.

What This Project Does
Loads diabetes prevalence data (BRFSS)
Cleans and filters for adult diabetes rates
Connects locations (state/county) with statistics
Builds a simple AI-powered chatbot (Google AI Studio / Python)
Returns diabetes prevalence for any U.S. location
